## 🔥 Proyectos destacados
### 🔹 [Shopping CART 🛒 (FullStack)](https://github.com/SuperGambaMan/shopping_cart)
**Shopping Cart** es una aplicación desarrollada en Java que implementa un sistema de carrito de compras simple, claro y extensible. Permite gestionar productos, añadirlos o eliminarlos del carrito y mantener el estado de la compra de forma estructurada. El proyecto está construido con Maven y organizado siguiendo buenas prácticas, lo que lo convierte en una base sólida para evolucionar hacia una aplicación completa de e‑commerce, integraciones con APIs o sistemas de inventario.

### 🔹 [Cafè Essenza ☕ (FrontEnd)](https://github.com/SuperGambaMan/Caf--Essenza--Proyecto-Final-DIW-)

**Cafè Essenza** es una página web completa diseñada como proyecto final para el módulo de Diseño de Interfaces Web (DIW). El objetivo del proyecto es presentar una experiencia visual elegante y moderna para una cafetería gourmet, combinando diseño atractivo, navegación intuitiva y contenido cuidadosamente estructurado. La web incluye secciones informativas, catálogo de productos, galería visual y elementos interactivos que mejoran la experiencia del usuario.
El proyecto destaca por su enfoque en la usabilidad, la coherencia estética y la aplicación de buenas prácticas de diseño responsive, asegurando que la web se vea y funcione correctamente en cualquier dispositivo.

### 🔹 [Ecommerce_jpa 🛍️ (BackEnd)](https://github.com/SuperGambaMan/ecommerce_jpa)

**Ecommerce_jpa** es una aplicación de comercio electrónico desarrollada en Java que utiliza JPA para la gestión de datos de forma eficiente y estructurada. El proyecto implementa las funcionalidades esenciales de un sistema e‑commerce, como la gestión de productos, usuarios y operaciones básicas de compra. Su arquitectura modular y su uso de buenas prácticas lo convierten en una base sólida para evolucionar hacia una plataforma más completa, integrando características como autenticación, pasarelas de pago o paneles de administración.
Es un proyecto ideal para demostrar dominio de Java, persistencia con JPA y diseño de aplicaciones orientadas a negocio.
